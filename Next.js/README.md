
# BullBearEye - TradeXray Dashboard (RTL Arabic, Next.js 13+)

This is a premium, RTL, Arabic-first crypto analytics dashboard built with [Next.js 13+](https://nextjs.org), TypeScript, and Tailwind CSS (with RTL and dark/light mode support). It features modular components, SaaS-ready structure, and is designed for institutional-grade analytics.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Features

- Arabic-first, RTL layout and design
- Modular components: Header, StatCard, SignalCard, WhaleTable, ThemeToggle
- Dark/Light mode toggle
- Tailwind CSS with RTL plugin
- Ready for SaaS and API integration
- Modern, institutional-grade UI/UX

## Quick Start

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Customization

- Edit `src/app/page.tsx` to change the dashboard layout or add new sections.
- Add or modify components in `src/components/`.
- Tailwind config is in `tailwind.config.js` (RTL and theme colors included).
